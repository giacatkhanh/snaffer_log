# Snaffler Formatter (snaffler-fmt) 🚀

A lightweight Python tool designed for penetration testers to parse, clean, prettify, and filter log files generated by **Snaffler** during Active Directory security assessments.

---

## ✨ Features
*   **Targeted Filtering:** Only care about high-value targets? Filter output to show only specific alert levels like `Red` or `Yellow`.
*   **ANSI Color Mapping:** Automatically converts textual color tags (`{Red}`, `{Yellow}`, `{Green}`) into true ANSI colors inside your terminal.
*   **Smart Unescaping:** Cleans up annoying `\r\n` lines, escape slashes, and format anomalies into structured code blocks.
*   **Dynamic Terminal Wrapping:** Detects your current terminal window width and automatically wraps long strings (like connection strings or configs).

---

## 🚀 Usage

**Basic Usage (Show all results):**
```bash
python3 snaffler-fmt.py snaffler.log | less -R
```

---

Advanced Usage (Filter by Colors):
You can specify which alert levels you want to see by passing them as a comma-separated list. For example, to only view Red (critical credentials) and Yellow (interesting databases) findings:
python3 snaffler-fmt.py snaffler.log Red,Yellow | less -R
